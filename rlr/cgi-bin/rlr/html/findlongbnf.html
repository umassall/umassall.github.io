WebSim

This applet checks all the files on the disk, finds all the WebSim-parsable files, and generates a BNF description of the language. This is the language for the code that is embedded in the Web page. The BNF is sent to standard out, which is the Java Console window in Netscape.

Back to WebSim.

This Java applet requires a Java-aware browser such as Netscape 2.0 for Solaris/Win95/WinNT.
