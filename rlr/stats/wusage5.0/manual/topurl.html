topurl

 
topurl          http://mysite/
The topurl option is used to specify a prefix for the links wusage will generate to the most popular documents in its reports. If this option is not present, wusage will create relative links, which work only if the reports are accessed via the web rather than as local files. To inhibit links entirely, use the nolinks option.
