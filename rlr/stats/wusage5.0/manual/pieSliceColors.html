The pieSliceColors option is used to specify the list of colors that will be used for each slice in the in the pie charts generated by Wusage. The first color is used for the first item in any pie chart, the second color for the second item, and so on. Wusage reuses colors when it reaches the end of the list.

You may specify as many colors as you want. Since there is a minimum size for a pie slice before the remainder of the items are placed in the "Other" category, Wusage will usually not display more than about eight slices.

Colors should be specified in the standard format used to specify all colors in HTML pages: 

RRGGBB
VERY IMPORTANT: do NOT place a # in front of the color, even though HTML would allow this, because this will cause Wusage to ignore that line as a comment. We apologize for this potential source of confusion.

Don't panic. there are plenty of pages on the web that provide lists of colors in this format, because it is also used in HTML pages.
