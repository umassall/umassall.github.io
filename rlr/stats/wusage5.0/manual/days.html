days

 
days on
days nochart
days off
If turned on, the days option indicates that wusage should report accesses to the server for each day of the week, month or summary period. This option automatically turns off if the reporting frequency is daily, since it is redundant in that case. The bar chart can be turned off by setting the days option to nochart. This option is useful to determine what days of the week or month produce the highest load on your server without the need to switch to daily reporting. The days option defaults to on. See also hours.
