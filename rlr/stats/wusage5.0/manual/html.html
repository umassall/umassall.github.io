html

 
html            3.0     (or 2.0)
The html option indicates what version of HTML wusage should output. In practice, if the html option is set to 3.0, wusage will produce HTML tables. These are attractive, but older browsers do not understand them. If the html option is set to 2.0, wusage will only produce lists, which are less attractive but understood by all browsers. If the html option is absent, wusage will assume HTML 3.0 is available.
