resultcodes

 
resultcodes on
resultcodes off
If turned on, the resultcodes option displays information regarding the most commonly reported HTTP protocol result codes. Possible result codes include 200 (a normal document request), 404 (document was not found), and 304 (document has not been modified and need not be sent again). This report can be turned off by setting the resultcodes option to off. The resultcodes report is turned on by default.
