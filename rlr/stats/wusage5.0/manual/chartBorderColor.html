chartBorderColor

 
#The color black, in HTML color format
chartBorderColor          FFFFFF
The chartBorderColor option is used to specify the outside border color for bar charts generated by Wusage. Colors should be specified in the standard format used to specify all colors in HTML pages: 
RRGGBB
Don't panic. there are plenty of pages on the web that provide lists of colors in this format, because it is also used in HTML pages.
