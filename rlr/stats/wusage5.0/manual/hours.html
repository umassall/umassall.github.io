hours

 
hours on
hours nochart
hours off
If turned on, the hours option indicates that wusage should report accesses to the server for each hour of the day. This report, which was always turned on in Wusage 4.1, can be turned off in Wusage 5.0 or higher by setting the hours option to off. The bar chart can be turned off by setting the hours option to nochart. This option is useful to determine when your network connection experiences the greatest load. The hours option defaults to on. See also days.
