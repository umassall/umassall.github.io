frequency

frequency       daily, weekly, monthly, or summary
frequency indicates how often wusage should output a report. If the frequency is daily, weekly or monthly, wusage will generate a report for every complete day, week or month. You can change the day on which a new week begins using the weekstart option. An index of these reports, with historical graphs, will be placed in the file index.html (or index.htm) in the report directory. The name of that file can be changed using the indexname option. If the frequency is summary, or the -s command line option is specified, then wusage will generate one overall summary report for the entire time period. In this case, the summary report will be placed in the file summary.html (or summary.htm) in the report directory. If the frequency option is not specified, weekly reports will be generated.
