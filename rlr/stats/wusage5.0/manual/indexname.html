indexname

indexname       index.html      (or another name)
The indexname option is used to change the name of the index page within the report directory. This option is only relevant when the frequency option is set to daily, weekly or monthly. Summary reports are always given the filename summary.html or summary.htm. If the indexname option is not given, the index of daily, weekly and monthly reports is named index.html or index.htm (the latter for DOS and related operating systems).
