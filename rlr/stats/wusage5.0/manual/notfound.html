notfound

notfound on
notfound nopie
notfound off
The notfound option, when it is present and not set to off, instructs wusage to produce a report listing the most common requests for documents that are not available on the server. If the notfound option is set to nopie, no pie chart is displayed. The number of documents not found to be listed can be controlled using the topnotfound option. The notfound option defaults to on.
