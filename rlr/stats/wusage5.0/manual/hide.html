The hide option is useful to keep certain documents from appearing in the popular-documents report. The hide option does not prevent documents from being counted toward total accesses, hourly throughput or any other report. To completely ignore a set of documents, see the ignore option instead. Documents that match any of the patterns listed for the hide option will be hidden from the top documents report. Each character of the document name is matched against the pattern. An * in the pattern matches any number of characters in the document name. Patterns have additional features; for more information about patterns, see the patterns section. Note that the suffixes option is applied first. If the hide option does not appear, all documents not filtered by the ignore option are candidates to appear in the popular documents report.
