This manual describes version 5.0. If you are using an older version, please consult the file manual.html, which is in the same directory with wusage itself. Version 5.0 includes this entire documentation set in the subdirectory manual.

The Macintosh version is very different. Macintosh users should consult only the documentation which comes with the Macintosh version.
