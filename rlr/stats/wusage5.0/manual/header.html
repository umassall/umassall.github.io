end header
The header option, which extends for several lines and must end with the line end header, is used together with the footer option to control the appearance of your reports. Any HTML tags and text found within the header option will be inserted at the beginning of the body of each HTML page wusage produces. The above example is found in the sample.conf file (sample.con under DOS) and in the configuration files produced by makeconf. If you open a <body> tag with the header option, be sure to close it with a </body> tag in the footer option. If the header ... end header option is absent, no HTML will be inserted. (The use of header is recommended for attractive results.)
