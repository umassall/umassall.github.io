trails

 
trails on
trails nopie
trails off
If present, the trails option indicates that wusage should output information about the trails (paths) that users followed while visiting the server. If the trails option is set to off, a trails report will not be generated. If it is set to nopie, the report is generated, but a pie chart is not drawn. This option is very useful to determine what routes individual users followed when visiting the site. See also toptrails, trailtimeout and trailminimumsteps.
