suffixes

 
suffixes
#Typical name for an index file
/index.html
#Also common
/home.html
#This merges requests both with and without a trailing /
/
end suffixes
The suffixes option, which extends for several lines and must end with the line end suffixes, indicates which suffixes will be removed from document names in order to combine references to the same document. Document names that end with any of the values listed in the suffixes option will be shortened. This can be used to combine references to, for instance, /index.html and /. Note: even if / is one of the suffixes listed, the first / in the document name will never be removed. Note that this option is applied before the allow and ignore options. If the suffixes ... end suffixes option is absent, no suffixes will be removed.
