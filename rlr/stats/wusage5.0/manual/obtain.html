How can I obtain wusage?

You can obtain an evaluation copy of wusage from Boutell.Com, Inc., by accessing the following URL:

http://www.boutell.com/wusage/download.html

We recommend that you access the above page to get the latest information about downloading wusage.

If you cannot access external http sites, just open an anonymous FTP session to:

ftp.boutell.com, directory: pub/boutell/wusage/

If a copy for your operating system is not available, feel free to contact us with information about your specific needs. (If the above link does not work for you, send email to wusage@boutell.com instead.)

After you download your evaluation copy and save it to disk, you will need to take the following steps to unpack it.

Unpacking wusage under Unix

Note that the filename of the archive you download may differ slightly to match your operating system's name: 

gunzip wusage5.0.osname.tar.gz
tar -xf wusage5.0.osname.tar
Note: version 4.1 was compressed using the Unix compress utility, not with gzip. Use uncompress rather than gunzip if you are working with an older platform for which version 5.0 is not yet available.

You are now ready to begin configuring wusage.

Unpacking wusage under MS-DOS, OS/2, Windows 95, and Windows NT

Using the popular pkunzip .zip file extraction program, cd to the directory where you plan to install wusage and type the following command at the DOS, OS/2, or Windows 95/NT console prompt: 

pkunzip /d wusage41.zip
Or use a user-friendly unzip program such as winzip, a product of Nico Mak Computing which we cheerfully recommend.

You are now ready to begin configuring wusage.

Your evaluation copy will work for 30 days, after which it will expire. See How do I register wusage (how much does it cost)? for information about acquiring a registered version.
