referrersinvisits

referrersinvisits on (or off)
The referrersinvisits option instructs wusage to include the referring URL, if any, in example visits and document trails. This makes it possible to find out where visitors from a particular web site typically go when they reach your site. This option is only available if your log contains referrer information, in addition to the normal common log format fields. By default, most servers do not log this information. See the server configuration tips section for more information.

The referrersinvisits option defaults to on if your server logs actually contain referrer information.
