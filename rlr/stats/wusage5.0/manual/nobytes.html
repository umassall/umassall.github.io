nobytes
The nobytes option indicates that "bytes transferred" should not be included in reports. Normally, Wusage reports on both individual accesses and the number of bytes transferred. Which of these two controls the sort order can be selected using the order option. If you are not interested in bytes-transferred information or others find it confusing, use the nobytes option to suppress it completely.
