ignorereferrers
#A pattern matching all referrers from our own site;
#useful to filter out internal links, making external
#links to our site stand out. Note the asterisk at the end!

http://www.boutell.com*
end ignorereferrers
The ignorereferrers option, which extends for several lines and must end with the line end ignorereferrers, indicates which referrer URLs will be ignored by wusage for purposes of generating the referrers report. Referring URLs which match any of the patterns listed for the ignorereferrers option will be ignored only for purposes of the referrers report; they are included normally in other reports. Each character of the referring URL is matched against the pattern. An * in the pattern matches any number of characters in the referring URL. Patterns have additional features; for more information about patterns, see the patterns section. If the ignorereferrers ... end ignorereferrers option is absent, no sites will be ignored.
