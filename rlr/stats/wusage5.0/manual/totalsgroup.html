totalsgroup
#Example: compare the total accesses to these
#three subdirectories, on a single 3D bar chart
/marketing*
/sales*
/development*
end totalsgroup
The totalsgroup option, which extends for several lines and must end with the line end totalsgroup, is used to declare a special interest in certain documents or sets of documents. Each line of the totalsgroup option contains a pattern and a title for that pattern. Wusage will count the total number of accesses to and bytes served of that pattern, and in the case of daily, weekly and monthly reports, a single bar graph comparing the popularity of those patterns over time will be generated. One combined bar chart is generated for all of the items. To present a separate bar chart for each item, see the totals option.

The three patterns shown above match all references to each of three different directory names. Each character of each document name is matched against each pattern in the totalsgroup list. An * in a pattern matches any number of characters in a document name. Patterns have additional features; for more information, see the patterns section.

The totalsgroup option may appear as many times as desired, in order to create several distinct groups, with one bar chart appearing per group. However, the same pattern should not appear more than once.
