weekstart

weekstart       sun, mon, tues, wed, thu, fri, or sat
The weekstart option indicates the day on which weeks are considered to begin. This is only significant when the frequency option is set to weekly. Only the first two letters of the day of the week are required. If this option does not appear, wusage assumes that a new week begins on Sunday.
