order

order           accesses        (or bytes)
The order option indicates whether reports, graphs and pie charts should be ordered by accesses or by bytes served. That is, if order is set to accesses, the most frequently accessed documents will appear first in reports. This is useful for counting visitors to a document. If order is set to bytes, the total number of bytes served in the course of delivering a document determines the sorting order. This is useful for determining the impact of a document on your network. If your server log does not contain information about the number of bytes transmitted, the order keyword is ignored. order defaults to accesses.
