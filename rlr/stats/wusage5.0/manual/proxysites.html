proxysites

proxysites on
proxysites nopie
proxysites off
If present and not set to off, the proxysites option indicates that wusage should output information about the sites that were accessed through your proxy server. This option is only useful if the server log you are analyzing came from a proxy server. "Proxy servers" are intermediate servers which are used to handle all of the outgoing World Wide Web access for an organization. See your server's documentation for more information about proxy server capabilities. If the proxysites option does not appear or is set to off, no proxy site statistics will be included in reports. If the proxysites option is set to nopie, no pie chart will be generated. See also topproxysites.
