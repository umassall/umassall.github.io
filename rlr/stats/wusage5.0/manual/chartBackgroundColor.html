chartBackgroundColor

 
#The color white, in HTML color format
chartBackgroundColor          FFFFFF
Since bar charts are transparent images, this color is rarely seen.

The chartBackgroundColor option is used to specify the background color for bar charts generated by Wusage. Colors should be specified in the standard format used to specify all colors in HTML pages: 

RRGGBB
Don't panic. there are plenty of pages on the web that provide lists of colors in this format, because it is also used in HTML pages.
