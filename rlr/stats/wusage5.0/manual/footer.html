The footer option, which extends for several lines and must end with the line end footer, is used together with the header option to control the appearance of your reports. Any HTML tags and text found within the footer option will be inserted at the beginning of the body of each HTML page wusage produces. Often, the footer option simply closes tags such as <body> opened in the header option. You may also wish to insert a link to your organization's home page. The above example is found in the sample.conf file (sample.con under DOS) and in the configuration files produced by makeconf. footer ... end footer option is absent, no HTML will be inserted. (The use of footer is recommended for attractive results.)
