sites

 
sites on
sites nopie
sites off
If turned on, the sites option indicates that wusage should output information about the sites which accessed the server. If the sites option does not appear or is set to off, no site-related or domain-related information will be included in reports. If the sites option is set to nopie, no pie chart will be generated. If your site is accessed by a wide variety of sites, you probably will want to turn off the pie chart, since no one site will represent a large enough portion of traffic to qualify for a slice. This option is useful to determine which sites accessed the server most often. The sites option defaults to off. See also dns, topsites, allowsites and ignoresites.
