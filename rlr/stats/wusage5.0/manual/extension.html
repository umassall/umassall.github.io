extension

 
extension          html
The extension option is used to specify the file extension for HTML files produced by Wusage. If this option is not present, Wusage defaults to htm under Windows NT and related environments, and html in other environments. This option is provided for the convenience of users who would prefer that Wusage produce files with a special extension, such as .shtml, which instructs some web servers to allow server-side includes. This can be useful in conjunction with the output customization options of Wusage.
