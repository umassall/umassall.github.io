topuseragents

topuseragents             10              (or another number, or all)
The topuseragents option indicates how many of the most frequently used user agents (web browsers) should be shown in each report. By default, the top ten user agents are shown. The sorting order is determined by the order option. You may specify any whole number of user agents or the word all.
