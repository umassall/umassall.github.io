Create a Forum

To create a forum, select the "Create a Forum" icon. When you click on this item, you will be presented with a form asking you for all the forum information:
  • Forum name: The full name of the forum, e.g. "Technical support Forum"
  • Code name to be appended to URL: This is what the new directory will be named, and consequently what will need to be entered in the forum's URL. Keep it short and lowercase.
  • Owner Code Name: This is the code name for the forum owner. NetForum comes with a default forum owner named "admin". If more persons will be owning forums in your site and you want to deleagate forum managing responsibilities, you will need to create their accounts within NetForum. To do this, go to the main menu and click on the "Manage Forum Owners" button.
  • Owner's Web Page: If this exists, the forum owner's name will always be displayed as a link to this site.
  • Forum Contact: The main contact person for the forum. An email address is required.
  • Contact's Email Address: The email address of the forum's contact.
  • Contact's Web Page: If this exists, the forum contact's name will always be displayed as a link to this site.
  • Can anyone add topics to this forum?; If no, then the user will be required a username and password before posting each new topic.
  • Forum Description: This description will be displayed when a user enters the forum. You may leave this blank.
  • Forum Guidelines: Guidelines specific to the forum; Delete the default text if this doesn't apply.
  • Forum group: This is a way of defining a group of individuals who's names and email addresses will always be accessible from Netforum. Enter one email address or name per line, alternating name/email address. If an individual has no email address, leave a blank line below their name. Delete the default text, or leave it as is, if this doesn't apply.

After entering the data, you may preview and edit the forum information as many times as you like before posting.

Once you click on "Post Forum", this is what happens:

  • A directory named whatever you entered in the "Code name to be appended to URL" field is created in the netforum cgi directory.
  • In this directory, two empty directories are created: topics and responses
  • In the forum directory, a file called "a" is created and made executable.
  • If you gave a forum description, a file called "description" is created in the forum directory.
  • If you gave forum guidelines, a file called "guidelines" is created in the forum directory.
  • If you defined a forum group, a file called "group" is created in the forum directory.
Your browser is then redirected to the new forum.
