NetForum Documentation

Administration and Maintenance

With NetForum 2.0 you can do forum maintenance within your web browser. If you are the NetForum administrator, you can perform all the functions that come with the admin script. Otherwise, you can only perform the maintenance functions that the NetForum administrator allowed you.

The following are the maintenance functions provided with NetForum:

  • Create a Forum Allows you to create new forums.
  • Edit Site Configurations Allows you to edit the NetForum system variables, the forums' default configuration, and the forums' default header and footer.
  • Edit Forum Allows you to edit a particular forum's information, delete a forum, configure a forum, and edit a forum's topics and messages.
    • Forum Configuration Allows you to configure an individual forum based on your personal preferences.
    • Edit Topics and Messages Allows you to edit a forum's topics and messages.
  • Manage Forum Owners Allows you to add, delete, and change forum owners.
